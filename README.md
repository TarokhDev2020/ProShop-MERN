
# ProShop-MERN

How to develope a fully complete ecommerce application with an admin panel to manage products


## Dependencies

 - axios
 - bcryptjs
 - dotenv
 - express
 - express-async-handler
 - jsonwebtoken
 - mongoose
 - morgan


## Installation

```http
  cd /Users/user/my_project
  git clone https://github.com/TarokhDev2020/ProShop-MERN.git
  npm install
```
